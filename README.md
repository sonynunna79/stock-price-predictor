# 📈 Stock Price Predictor

This is a simple AI project that predicts stock prices using Machine Learning.

## 🔧 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## ▶️ How to Run

1. Install libraries:
pip install pandas numpy scikit-learn matplotlib

2. Run the project:
python main.py

## 📊 Output
- Predicts next 10 days stock prices
- Displays graph
